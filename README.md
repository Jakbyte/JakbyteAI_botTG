# 🤖 Jakbyte_AI — Асинхронний Telegram-бот з інтеграцією ШІ

Асинхронний Telegram-бот, побудований на сучасній бібліотеці **Aiogram 3**, який підключений до моделей штучного інтелекту через **OpenRouter API**. Бот має чисту модульну архітектуру та захищений від витоку API-ключів за допомогою змінних середовища.

---

## 🚀 Функціонал
* **⚡ Асинхронність:** Повна підтримка асинхронних запитів, бот не блокує потік і може обслуговувати багатьох користувачів одночасно.
* **🧠 ШІ Інтеграція:** Робота з моделлю `gpt-4o-mini` (або іншими на вибір) через OpenRouter.
* **👁️ Візуальний відгук:** Ефект `typing...` (Бот друкує) під час генерації відповіді від ШІ.
* **📱 Зручне меню:** Вбудована зручна Reply-клавіатура для швидкого доступу до команд.
* **🛡️ Безпека:** Усі чутливі токени винесені в конфігураційний файл `.env` та заігнорені в Git.

---

## 🛠️ Архітектура проєкту

Проєкт розділено на модулі для запобігання циклічних імпортів та легкого масштабування:

* `main.py` — Головна точка входу, яка запускає бот-полінг та містить хендлери (обробники команд).
* `create_bot.py` — Ізольований модуль ініціалізації об'єктів бота (`Bot`, `Dispatcher`) та клієнта ШІ (`AsyncOpenAI`).
* `.env` — Конфіденційні ключі авторизації (не завантажуються в репозиторій).
* `requirements.txt` — Список залежностей проєкту.
